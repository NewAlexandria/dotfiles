# Ghostty & Cmux Theme Management

## How to Change Terminal Themes
You can change your Ghostty and Cmux theme by editing `~/.dotfiles/ghostty/config`.

### Recommended Themes:
- `dracula`
- `catppuccin-mocha`
- `ayu-mirage`
- `nord`

Example:
```bash
# Edit this file:
nano ~/.dotfiles/ghostty/config

# Add or update the theme line:
theme = dracula
```

## How to Change Gemini CLI Theme
You can change the `gemini-cli` theme from within the CLI itself by running:
```/theme```
Or by editing `~/.gemini/settings.json`.
